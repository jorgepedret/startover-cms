# Template Page Structure

This template shows how to structure pages in Sveltia CMS. The frontmatter above contains all the metadata and content sections. Replace the values with your actual content.

## To Create a New Page:

1. Click "New" in Sveltia CMS
2. Fill in the fields:
   - **Site Slug**: URL-friendly identifier (no spaces, lowercase)
   - **Title**: The page heading
   - **Intro Text**: Opening paragraph
   - **Hero Image**: Main visual (upload from Media library)
   - **Content Sections**: Add as many sections as needed with title, content, and optional images
   - **Tags**: For organization and filtering
   - **Status**: Draft or Published

3. Save and publish when ready
